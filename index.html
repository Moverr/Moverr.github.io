Polymorphic Functions vs Monolythic Functions

Polymorphic functions are gneric in nature, what this generally means, you dont have to specify the data type of the input. This is helpful in functional programming, it helps a function to serve a number of use cases without changing state.

Example of Monolythic Function.
def add(x:Int,y:Int):Int={ x+y }
Utilizing such a function would require you to pass only integers otherwise it would throw an exception
Take a look at example 2.
def add[T](x:T,y:T):T={ x+y }
The above code snippet can take on params of any type, say ints, doubles, floats and the code will work well.
